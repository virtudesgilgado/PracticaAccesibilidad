Formulario de Registro

Complete el siguiente formulario para registrar sus datos. Los campos marcados con * son obligatorios. Puede navegar entre los campos usando la tecla Tab para avanzar y Shift + Tab para retroceder.

Instrucciones de uso

  • Use Tab para moverse al siguiente campo y Shift+Tab para volver al anterior.
  • Los campos obligatorios están marcados con un asterisco (*) y un borde rojo a la izquierda.
  • Las listas desplegables se abren con Espacio o Enter y se navegan con las flechas ⬆ ⬇.
  • Las casillas de verificación se marcan y desmarcan con Espacio.
  • Los botones de radio se navegan con las flechas ⬆ ⬇ ⬅ ⮕.
